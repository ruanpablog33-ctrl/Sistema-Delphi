## 📝 Sobre o Projeto
O **Sistema de Controle de Finanças** é uma aplicação desktop desenvolvida em Delphi focada na gestão financeira pessoal e empresarial. O objetivo do sistema é facilitar o registro, monitoramento e análise de receitas e despesas, garantindo um controle de fluxo de caixa eficiente, organizado e seguro, utilizando o MySQL como sistema de gerenciamento de banco de dados relacional.

## ✨ Funcionalidades Principais
- **Gestão de Transações:** Cadastro, edição e exclusão de receitas e despesas.
- **Categorização:** Classificação de gastos e ganhos por categorias (ex: Moradia, Alimentação, Salário).
- **Controle de Fluxo de Caixa:** Saldo atualizado em tempo real.
- **Relatórios:** Geração de relatórios financeiros filtrados por período (diário, mensal, anual).
- **Autenticação:** Sistema de login para segurança das informações financeiras.

## 🛠️ Tecnologias e Ferramentas Utilizadas
- **Linguagem:** Object Pascal (Delphi)
- **IDE:** Embarcadero Delphi (RAD Studio 13)
- **Banco de Dados:** MySQL
- **Componentes de Acesso a Dados:** FireDAC

## ⚙️ Pré-requisitos
Antes de executar o projeto, você precisará ter instalado em sua máquina:
* [MySQL Workbench](https://dev.mysql.com/downloads/workbench/) ou DBeaver (Opcional, para gerenciamento do BD)
* IDE Delphi compatível (https://www.embarcadero.com/br/products/delphi)
